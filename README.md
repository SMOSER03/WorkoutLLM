# WorkoutLLM

This repository contains the source code for the WorkoutLLM bachelor thesis prototype.

## Setup

First, clone this repository:

```bash
git clone <repository-url>
cd <repository-folder>
```

The project consists of two separate parts:

* **Frontend**: Located in the `WorkoutLLM` folder
* **Backend**: Located in the `backend` folder

To set up and run the project, follow the instructions provided in the corresponding README files:

### Frontend

Navigate to the frontend folder:

```bash
cd WorkoutLLM
```

Follow the setup instructions described in:

```
WorkoutLLM/README.md
```

### Backend

Navigate to the backend folder:

```bash
cd backend
```

Follow the setup instructions described in:

```
backend/README.md
```

Both parts need to be configured and running according to their respective README instructions for the complete application to work.
